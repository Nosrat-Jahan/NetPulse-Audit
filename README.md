# 🛡️ NetPulse-Audit | Network Intelligence Suite

**NetPulse-Audit** is a sophisticated telemetry engine designed for real-time network reconnaissance and system auditing. It bridges the gap between hardware abstraction and network security by providing a transparent view of data transmission paths and active process engagement.

## ⚡ Core Intelligence
- **Active Node Mapping**: Real-time extraction of established TCP/UDP sockets and remote endpoints.
- **Process Telemetry**: Dynamic tracking of local-to-remote IP paths and associated PIDs.
- **Heuristic Integrity Sweep**: Automated system resource analysis to identify anomalous load spikes.
- **Cyber-Industrial Dashboard**: A zero-latency, async-driven interface for continuous system observation.

## 🛠️ Technology Stack
- **Backend:** Python 3.14 / Flask (REST API v1)
- **Telemetry Engine:** Psutil (Hardware Abstraction)
- **Frontend:** HTML5, Modern CSS (Inter & JetBrains Mono), JavaScript (Async/Await Fetch)

## ⚙️ Installation
1. `git clone https://github.com/Nosrat-Jahan/NetPulse-Audit.git`
2. `pip install -r requirements.txt`
3. `python netpulse_audit.py`
